Good-Sees:
a website like goodreads but with movies

I was the primary middleware developer for Good-Sees. As the middleware developer I was responsible to link up all the data passage from the front end to the backend. This included where we would hit APIs and our own database for data, getting the data to pass from the backend to the front end to be displayed. Although I imitated goodreads, I wanted to make Good-Sees more clean and simple for the user to navigate.

MVMT Clone

I decided to clone MVMT.com because I liked their products and site design. I completed all work from the frontend to the backend, learning how to build a site that provides a unique but beautiful design for the user. MVMTclone!
